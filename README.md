# Ticket-Master-Dashboard-Analysis
Ticket Master Dashboard is a data analytics project using SQL and Power BI to analyze support tickets. It provides insights on ticket volume, resolution time, priority, queues, and trends, helping optimize support performance, improve SLA management, and enable better decision-making through data-driven insights.

A complete data analysis project using Power BI and SQL to analyze customer support tickets.

## 📊 Features
- Total Tickets Overview
- Avg Resolution Time
- Ticket Distribution by Type, Priority, Queue
- Monthly Trends
- Tag-based Analysis
- Geo Mapping of Tickets

## 🛠 Tech Stack
- Power BI
- SQL (MySQL/PostgreSQL)
- Excel

## 📈 Insights
- Most tickets are Technical & Security related
- Medium priority tickets dominate
- Technical Support queue handles majority load
- Peak ticket months: Jan–April

## 🧠 Use Cases
- Customer Support Optimization
- SLA Monitoring
- Resource Allocation

## 📂 Dataset
Dataset includes:
- Ticket ID
- Date (Created & Resolved)
- Queue
- Priority
- Type
- Tags (Primary, Secondary, Category)

## 🧾 SQL Analysis
Check `/sql` folder for:
- Basic queries
- Advanced business insights
